# Project Name
ArrowNU

## Description
Web-based navigation game focused on educating the user/s about the Northeastern University Boston campus.

## Installation
```bash
npm install
```

## Usage
```bash
npm start
```

## Features
- Party creation system with a team leader and player
- Key destinations that you get to travel to and learn about
- Satvik Singh's github portfolio full access

## Technologies
- Node.js
- React
